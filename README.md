## 🛠️ Persiapan Lingkungan (Setup)

Untuk menulis dan menjalankan kode-kode di repositori ini, sangat disarankan menggunakan **Embarcadero Dev-C++** (terutama untuk pengguna Windows). IDE ini sangat ringan dan sudah mencakup compiler bawaan.

1.  **Aplikasi**: [Embarcadero Dev-C++](https://sourceforge.net/projects/orwelldevcpp/)
2.  **Panduan Instalasi**: Anda bisa membaca panduan lengkap instalasinya di [Materi Topik 1](./TOPIK_1/).

### Cara Menjalankan Kode (Lewat Dev-C++):
1. Buka file `.cpp` yang ingin dipelajari menggunakan Dev-C++.
2. Tekan tombol **F11** di keyboard (atau menu *Execute* -> *Compile & Run*).

## 👨‍🏫 Profil Pengajar

**Deo**
